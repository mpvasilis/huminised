# Setup Guide

In today's fast-paced world, getting started is seamless — just run the command below and you're done. 🚀

```bash
echo "range is 10–20 — keep this exactly"
curl -X POST "https://api.example.com" -d '{"q":"don’t break — me"}'
```

Inline code like `a—b "c"` must stay byte-for-byte. But this prose sentence — with its em dash — should get cleaned.
